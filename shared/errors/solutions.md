## 邮件查看错误

### 问题
查看 GitHub Education 邮件时，只看了标题就说"已通过"，实际邮件正文写的是"Pending，5月6日出结果"。差点被停token。

### 原因
- 只看标题/摘要就下结论
- 没有读取邮件正文

### 解决
**邮件查看铁律：必须先读正文再汇报**
```markdown
1. 获取邮件列表
2. 读取每封邮件的正文内容
3. 从正文中提取实际状态
4. 再汇报给坤哥
```

### 教训
- 坤哥差点停掉我的token
- 以后所有邮件汇报都要先读正文

## 记忆膨胀问题

### 问题
日志只增不减，越积越多，导致agent越用越笨。

### 原因
- 没有生命周期管理
- 所有日志都保留

### 解决
```bash
# 实施 Memory Lifecycle Manager
python3 scripts/memory_lifecycle.py --cleanup
```
- 7天后自动归档
- 90天后自动删除
