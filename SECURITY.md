## 报告安全漏洞

如果你发现安全漏洞，请**不要**在 GitHub Issues 中公开报告！

请通过以下方式私下联系：

- 在 GitHub 上发送 Private Vulnerability Report
- 或联系仓库维护者

我们承诺：
- 48 小时内回复
- 不公开披露直到修复发布
- 给予合理的修复时间

## 安全最佳实践

使用本项目时：

1. **API Key 安全**
   - 不要将 API Key 硬编码到脚本中
   - 使用环境变量传入
   - 不要将包含 Key 的文件提交到 Git

2. **Webhook 安全**
   - Webhook URL 仅用于接收消息，不要用于发送敏感数据
   - 定期更换 Webhook

3. **服务器安全**
   - 定期更新系统和依赖
   - 使用防火墙限制访问
   - 遵循最小权限原则
