# 💬 获取支持

## 寻求帮助

在寻求帮助之前，请先：

1. 📖 阅读 [README.md](./README.md) 了解项目概述
2. 🔍 搜索 [已有的 Issues](https://github.com/YuchenKuney/openclaw-memory-architecture-public/issues) 看是否已有类似问题
3. 📚 查看 [Wiki](https://github.com/YuchenKuney/openclaw-memory-architecture-public/wiki)（如有）

## 获取支持的方式

### GitHub Issues（推荐）

- 🐛 Bug 报告 → [新建 Bug Issue](https://github.com/YuchenKuney/openclaw-memory-architecture-public/issues/new?template=bug_report.md)
- 💡 功能建议 → [新建 Feature Issue](https://github.com/YuchenKuney/openclaw-memory-architecture-public/issues/new?template=feature_request.md)

### 社区讨论

- 💬 使用 GitHub Discussions 进行一般性讨论
- ❓ 在 Q&A 分类中提问

## 响应时间

- Bug 报告：通常 1-3 天内回复
- 功能建议：通常 1 周内回复
- 一般问题：通常 3-5 天内回复

## 注意事项

⚠️ 请勿在公开渠道分享敏感信息（API Key、密码等）

感谢你的耐心！🙏
